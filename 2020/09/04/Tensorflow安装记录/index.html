Tensorflow安装记录

Posted on 2020-09-04 Edited on 2020-09-27 In 日常记录 Views:

在anaconda下安装tensorflow

电脑不是NVIDIA或者AMD独显，是因特尔集显，所以不需要考虑CUDA，安装CPU版本即可。

conda版本：conda 4.5.1

python 版本：3.7.1

conda -list 可以查看所有包的版本信息

第一次尝试

参考简书上一篇博客：anaconda+tensorflow安装完整步骤

没有创建新环境，直接pip install --upgrade --ignore-installed tensorflow

但报错了，应该是权限的问题，所以加上--user pip install --upgrade --ignore-installed tensorflow--user

但安装过程中仍有错误信息（但python版本明明是3.7）：

还有一些warning：

但最终显示successfully installed。

尝试import tensorflow报错：

网上搜索发现有很多人和我遇到了同样的问题，说是安装的版本和自己的环境不匹配，但网上都是建议安装到1.x版本的，而我需要安装2.0以上的版本。

查看安装的tensorflow的版本：

第二次尝试

  1. 先卸载pip uninstall tensorflow
  1. 安装高版本pip（安装了20.2.2）
  1. 安装tensorflow

pip install tensorflow==2.0.0

  1. import tensoflow成功
  2. 之后可能会出现numpy的错误，此时用pip升级numpy即可

到这里就结束啦！

注意：清华大学开源软件镜像站上tensorflow windows的镜像已废弃，新版本在PyPI上。
