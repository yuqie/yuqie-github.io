Xming+XShell

Posted on 2019-09-06 In Text Views:

之前Xshell+Xming用的好好的，最近总是报错localhost:19.0，Xshell中已经设置localhost:0.0

解决方案：重新配置XShell + Xming

step1. XMing的配置：打开XLaunch，记住Display Number，现在这里是0，后面配置及XShell中会用到。然后点击下一步启动xming.

step2. 首先打开Xming安装文件夹找到 X*.hosts 文件(*号为上面Display Number 数字），如本例就是找到X0.hosts文件，打开并在localhost下面一行，添加Linux服务器的IP地址。

step3. 配置xshell,连接-》SSH -> 隧道-》在x11 转移中设置xdisplay为localhost:0.0

step4, 配置完成后，重新连接服务器，

启动图形界面软件，gnuplot 就可以在xming上启动了。
