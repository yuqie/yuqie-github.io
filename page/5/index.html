【转载】如何优雅地绘制布里渊区？

Posted on 2018-09-17 Edited on 2019-07-21 In Text

布里渊区（Brillouin zone）是固体物理学中一个十分重要的概念，我们在写论文时也经常需要绘图展示研究材料的布里渊区。但布里渊区是一个三维图，很难用二维的平面绘图软件直接绘制。

Xcrysden（http://www.xcrysden.org/）这个软件可以在看晶体结构的同时输出布里渊区矢量图，而Adobe Illustrator（以下简称AI）是矢量绘图的专业软件之一。如果两者结合在一起，就可以绘制出布里渊区，而且图片也很美观。

软件：Xcrysden、AI

步骤：

1. 用Xcrysden打开晶体结构文件（xsf格式或pwscf、Wienk结构文件），选择菜单“Tools”→“ k-path Section”，如下图所示。

![](https://image.sciencenet.cn/home/201805/04/195038fbvep5kb8u8e75kp.png)

2. 在弹出的“Band Path Section”窗口中选择要标注的k点，其他点可以点击不显示。

![](https://image.sciencenet.cn/home/201805/04/195134i2xj6x9lqe29d09l.png)

3. 用鼠标拖动布里渊区，调整好角度后点击左边的打印机按钮，在弹出的对话框中选择“Print to”→“file”，并填写好相应的Filename（xxx.eps）后点击打印。

![](https://image.sciencenet.cn/home/201805/04/195053bg9qxj2xp2pzjaqk.png)

4. 将生成的eps文件用AI软件打开(而非拖动到已有的AI文件中)，依次双击左边的图标和上面的字并删除。

![](https://image.sciencenet.cn/home/201805/04/195436l7qp1obbjb4pnbbt.png)

5. 由于布里渊区是矢量图，双击线可以进行编辑。然后在图片上添加必要的线，标出必要的文字和图标，再和其他图片组合在一起，保存文件并导出图片。

最后效果如图：

![](https://image.sciencenet.cn/home/201805/04/195855my926lo5zyb4dyza.png)

说明：

1. Xcrysden软件可能需要在Linux系统下安装；

2. 布里渊区的位置和名称可以参考网站：http://www.cryst.ehu.es/cryst/get_kvec.html

或文章：https://www.sciencedirect.com/science/article/pii/S0927025610002697?via%3Dihub 

3.   “Γ”等希腊字母在AI的标注：选择菜单“文字”→“字形”，点击需要输入特殊字符。

“窗口”→“文字”→“字符”，点击右边的显示选项按钮后点击下标（高版本可以直接显示下标按钮）。

使用MATLAB将BXSF文件转换为可以用VESTA打开的XSF文件

Posted on 2018-06-04 Edited on 2019-07-21 In Text

http://blog.sciencenet.cn/blog-1502061-1026133.html

第一性原理软件生成费米面（Fermi surface）标准文件是BXSF（Band-XSF）格式的，BXSF文件可以用XCrySDen软件打开。但是XCrySDen绘图不是很漂亮，而VESTA绘图要比XCrySDen美观很多。那是否可以将BXSF文件放在VESTA里绘图呢？答案是否定的。这是由于BXSF和XSF文件格式有差别导致的。

   BXSF和XSF在文件格式上的差别主要有两个。第一、BXSF文件包含多条能带的数据，也就是多套网格数据，而XSF文件只有一个网格的数据。第二、BXSF和XSF的数据排列正好相反，XSF文件的数据按照“column-major (i.e. FORTRAN) order”的顺序排列网格（datagrid）数据的（如下图1），而bxsf文件的数据按照“row-major (i.e. C) order”顺序排列能带网格（bandgrid）数据的（如下图2）。具体文件格式可以参见XCrySDen网站介绍(http://www.XCrySDen.org/doc/XSF.html)。

![](https://image.sciencenet.cn/album/201701/07/204244kjy5nkynxxnjwg48.png)

图 1 xsf文件的数据顺序：the values inside a datagrid are specified in column-major (i.e. FORTRAN) order

![](https://image.sciencenet.cn/album/201701/07/204302kztzswjrtgwtgitg.png)

图 2  bxsf文件的数据顺序：the values inside a bandgrid are specified in row-major (i.e. C) order.

  这里提供一个MATLAB程序bxsf2xsf.m将BXSF文件按照XSF的顺序重新排列，同时将每一条能带数据分别生成一个XSF文件。

所需程序：MATLAB、VESTA

脚本和例子： ![](https://blog.sciencenet.cn/static/ueditor/dialogs/attachment/fileTypeImages/icon_rar.gif) bxsf2xsf.zip(点击下载)

使用方法：

   一般情况下，BSXF文件里的能带未必都穿过费米面，因此需要修改bxsf2xsf.m脚本里下面两个参数：

这两个参数设置穿过费米面能带的序号（BXSF出现的顺序从1开始，而非BAND后面的数字）。按F5运行bxsf2xsf.m，按照提示在弹出的“选择要打开的文件”对话框中选择要处理的BXSF文件。运行成功后就可以得到一批“BXSF文件名+.band#.xsf”的文件，其中band后面的数字对应BXSF文件里能带的序号。这个脚本已经将能量都减去了Fermi能级，因此生成的XSF文件的费米能级为0。

 对于网格较稀疏BXSF文件，用bxsf2xsf.m处理可能会导致绘制的费米面出现孔洞和毛刺的现象，可以使用bxsf2xsf_inter.m脚本处理（见附件），类似XCrySDen的Interpolation功能，这个脚本使用内插的方法得到较密的能带网格点。内插倍数就是网格增加为原来多少倍。内插倍数建议2-5倍，更高的倍数可能导致生成的XSF文件较大。

绘图步骤 （适用于VESTA version 3）

1. 先打开一个*.band#.xsf文件，然后再载入其他的XSF文件，方法：选择菜单Edit → Edit Data → Volumetric Data…在弹出的对话框中选择Import…，选择其他的XSF文件，然后选择Multiply to current data，如图3所示，最后点击应用扭。

![](https://image.sciencenet.cn/album/201701/07/204411j31d17806xgxu8gs.png)

图 3 将*.band#.xsf文件导入到VESTA中

2. 点击软件主界面左下角的Properties…按钮，在弹出的对话框中将Isosurface选项卡的Isosurface level值调为0，同时Sections选项卡里的Opacity of drawn secions的值也改为0，如图4所示。

![](https://image.sciencenet.cn/album/201701/07/204439gjapwpptxuguwz5l.png)

图 4  将Isosurface level值调为0，同时将Opacity of drawn secions的值也改为0

最终效果如下图5所示。可以将文件保存为*.vesta格式的文件方便以后查看。

![](https://image.sciencenet.cn/album/201701/07/204509o0m0ma0hmqa0wl0h.png)

图 5  最终效果图（MgB2的Fermi surface）

说 明：

1.虽然用VESTA绘制的费米面比XcrySDen美观，但是也有下面几个缺点：(1)只能在倒空间原胞（而非倒空间布里渊区）内绘制；(2)费米面颜色只能统一设定，而非像XCrySDen那样可以单独设定每一个费米面的颜色。如果有网友能解决上面两个问题请@我。

2.用VESTA可以很方便地绘出沿任意切面的费米面。在Utilities菜单中选择2D Data Display…，并设置合适的切面参数就可以绘制像图6一样的效果。

3.使用Volumetric data的Surface coloring功能还可以在费米面上加上颜色，比如可以绘制轨道分辨的费米面，如图7所示。
